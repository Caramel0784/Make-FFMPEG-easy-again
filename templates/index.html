    Custom Command

    For anything not covered above. Type ffmpeg arguments as you would on the command line (without the word "ffmpeg" itself). Use {IN} and {IN2} as placeholders for uploaded files, and {OUT} for the output. Example:
    -i {IN} -vf "eq=brightness=0.1" {OUT}
